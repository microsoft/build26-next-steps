# Global Skilling

Worldwide skilling programs and learning paths from Microsoft Build 2026.

## Regional Learning Programs

### Microsoft Learn
- [Microsoft Learn](https://learn.microsoft.com/) — Free, interactive, hands-on training across all Microsoft technologies
- [Microsoft Learn for Organizations](https://learn.microsoft.com/training/organizations/) — Enterprise skilling programs and team learning paths
- [Microsoft Learn Student Hub](https://learn.microsoft.com/training/student-hub/) — Resources and learning paths for students

### Cloud Skills Challenges
- [30 Days to Learn It](https://developer.microsoft.com/offers/30-days-to-learn-it) — Self-paced challenges across Azure, Microsoft 365, Power Platform, and more
- [Applied Skills credentials](https://learn.microsoft.com/credentials/browse/?credential_types=applied%20skills) — Scenario-based assessments that validate practical skills

### Community-Led Learning
- [Microsoft Reactor](https://developer.microsoft.com/reactor/) — Free, in-person and virtual workshops hosted worldwide
- [Microsoft Tech Community](https://techcommunity.microsoft.com/) — Forums, blogs, and discussions with peers and Microsoft experts
- [Microsoft User Groups](https://developer.microsoft.com/community/user-groups/) — Local developer communities around the world

## Localized Resources

- [Microsoft Learn — Language support](https://learn.microsoft.com/locale/) — Training content available in 20+ languages
- [Microsoft Regional Developer Centers](https://developer.microsoft.com/) — Regional events, hackathons, and learning programs

## Additional Resources

- [GitHub Education](https://education.github.com/) — Free developer tools for students and educators
- [Microsoft Imagine Academy](https://www.microsoft.com/education/imagine-academy) — Technology education programs for academic institutions
