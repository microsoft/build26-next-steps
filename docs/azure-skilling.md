# Azure Skilling

Azure-focused learning plans and skilling resources to continue your Microsoft Build 2026 journey.

## Learning Paths

### Getting Started with Azure
- [Azure Fundamentals (AZ-900)](https://learn.microsoft.com/training/paths/az-900-describe-cloud-concepts/) — Cloud concepts, Azure services, security, privacy, and pricing
- [Azure AI Fundamentals (AI-900)](https://learn.microsoft.com/training/paths/get-started-with-artificial-intelligence-on-azure/) — Core AI and machine learning concepts on Azure
- [Azure Developer Associate (AZ-204)](https://learn.microsoft.com/training/paths/create-azure-app-service-web-apps/) — Build, test, and deploy applications on Azure

### AI & Agents
- [Build AI apps and agents with Microsoft Foundry](https://learn.microsoft.com/training/paths/create-custom-copilots-ai-studio/) — End-to-end AI application development
- [Develop AI agents using Azure OpenAI and Semantic Kernel](https://learn.microsoft.com/training/paths/develop-ai-agents-azure-open-ai-semantic-kernel-sdk/) — Agent development patterns and frameworks
- [Microsoft Copilot development](https://learn.microsoft.com/training/browse/?terms=copilot) — Build and extend Copilot experiences

### Cloud-Native Development
- [Azure Kubernetes Service (AKS)](https://learn.microsoft.com/training/paths/intro-to-kubernetes-on-azure/) — Deploy and manage containerized applications
- [Azure Container Apps](https://learn.microsoft.com/training/paths/deploy-cloud-native-applications-to-azure-container-apps/) — Serverless container hosting
- [.NET Aspire for cloud-native apps](https://learn.microsoft.com/training/paths/dotnet-aspire/) — Build distributed .NET applications

### Data & Analytics
- [Azure SQL for developers](https://learn.microsoft.com/training/paths/azure-sql-fundamentals/) — Relational database development on Azure
- [Azure Cosmos DB developer](https://learn.microsoft.com/training/paths/build-dotnet-applications-with-azure-cosmos-db/) — NoSQL database for modern applications
- [Microsoft Fabric analytics](https://learn.microsoft.com/training/paths/get-started-fabric/) — End-to-end analytics platform
