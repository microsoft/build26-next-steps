# AI Skills Fest

AI Skills Fest learning programs and resources from Microsoft Build 2026.

## What is AI Skills Fest?

AI Skills Fest is a free, global learning event that helps developers, IT professionals, and data scientists build practical AI skills through hands-on challenges and guided learning paths.

## Learning Challenges

### Azure AI & Foundry
- [Build intelligent apps with Azure AI services](https://learn.microsoft.com/training/paths/create-custom-copilots-ai-studio/) — Hands-on path for building AI applications with Microsoft Foundry
- [Get started with Azure OpenAI Service](https://learn.microsoft.com/training/modules/get-started-openai/) — Core concepts and first projects with Azure OpenAI

### GitHub Copilot
- [Introduction to GitHub Copilot](https://learn.microsoft.com/training/modules/introduction-to-github-copilot/) — Learn how to use GitHub Copilot to accelerate your development
- [Accelerate app development by using GitHub Copilot](https://learn.microsoft.com/training/paths/accelerate-app-development-using-github-copilot/) — Intermediate workflows and agent mode patterns

### Agents & Automation
- [Develop AI agents using Azure OpenAI and Semantic Kernel](https://learn.microsoft.com/training/paths/develop-ai-agents-azure-open-ai-semantic-kernel-sdk/) — Build intelligent agents that reason, plan, and act
- [Microsoft Copilot Studio](https://learn.microsoft.com/training/paths/work-power-virtual-agents/) — Low-code agent development for business scenarios

### Responsible AI
- [Responsible AI principles and practices](https://learn.microsoft.com/training/paths/responsible-ai-business-principles/) — Build AI systems that are fair, reliable, and transparent

## How to Participate

1. Visit [AI Skills Fest](https://aiskillsfest.com) for the latest challenge schedule
2. Choose a learning path that matches your interests and skill level
3. Complete the challenges and earn digital badges
4. Share your progress with the community
